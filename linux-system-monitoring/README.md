# Linux System Monitoring & Process Analysis

## Objective
Monitor and analyze system performance using Linux commands to understand CPU usage, memory consumption, and running processes.

## Tools Used
- Linux
- Terminal Commands

## Commands Used
- top
- ps aux
- df -h
- free -m
- uptime

## Analysis Performed
- Reviewed active processes and CPU utilization
- Checked memory usage
- Monitored disk space usage
- Evaluated system uptime

## Key Findings
- Identified processes consuming the most CPU resources.
- Reviewed available and used memory.
- Verified available disk space to ensure system stability.
- Observed system uptime to evaluate reliability.

## Skills Demonstrated
- System monitoring
- Command-line proficiency
- Performance analysis
- Troubleshooting fundamentals
