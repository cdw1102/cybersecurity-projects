# Student Performance Data Analysis

## Objective
Analyze student performance data to identify trends, calculate key statistics, and visualize results.

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
Simulated dataset of student test scores.

## Analysis Performed
- Calculated average score
- Identified highest and lowest scores
- Determined pass/fail rates
- Generated a basic visualization of score distribution

## Skills Demonstrated
- Data cleaning
- Statistical analysis
- Basic data visualization
- Python programming
